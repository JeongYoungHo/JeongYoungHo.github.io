jekyll의 버전 업그레이드로 인해서 theme가 기본적으로 입혀져 있다고 한다.
그래서 디렉토리에서 보면 `_layouts, _includes...`가 기본적으로 보이지 않는다.
[Jekyll docs theme][jekyll-docs-theme]에서 해당 내용을 확인하고 해결하였다.
